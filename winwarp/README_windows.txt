Hidr4lisk_WARP — Windows
========================

SETUP
-----
1. Go to  hidr4lisk.github.io/warp
2. Generate a session → download warp.py
3. Place warp.py in the same folder as Hidr4lisk_WARP_windows.exe
4. Double-click Hidr4lisk_WARP_windows.exe

USAGE
-----
CHAT   → Type in the input bar, press Enter or click SEND
FILES  → Drag files into the PORTAL zone, or click [BROWSE]
         Received files are saved in the "received/" subfolder
SHARE  → Send your warp.py to your peer — they do the same

COMPATIBILITY
-------------
WinWarp is fully compatible with the terminal version (warp.py).
One side can use the GUI, the other the terminal — same session.
